# Screenshots Directory

Place your application screenshots here. Recommended screenshots:

1. **main-interface.png** - Main application window showing the drag-and-drop zone
2. **download-progress.png** - Active downloads with progress bars
3. **settings.png** - Settings panel

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or 1280x720
- **Naming**: Use descriptive names like `main-interface.png`, `download-progress.png`, etc.
- **Quality**: Use high-quality screenshots that showcase the UI clearly

## How to Take Screenshots

1. Run the application: `npm start`
2. Take screenshots of different views:
   - Main window with drag-drop zone
   - Active downloads in progress
   - Settings panel
   - Completed downloads view
3. Save them in this directory with the names referenced in README.md
