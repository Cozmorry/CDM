# Assets Folder

This folder contains the application icons and logo.

## Required Files

1. **icon.png** (or icon.ico for Windows) - Application icon
   - Recommended sizes: 256x256, 512x512, 1024x1024
   - Used for: Window icon, taskbar icon, app icon

2. **logo.png** - Application logo
   - Recommended size: 32x32 to 64x64
   - Used for: Header logo in the application

## Icon Formats

- **Windows**: `.ico` file (multi-resolution)
- **macOS**: `.icns` file
- **Linux**: `.png` file

For Windows builds, you can use `icon.ico` which will be automatically used by electron-builder.

## Creating Icons

You can create icons using:
- Online tools: https://www.icoconverter.com/
- Image editing software (Photoshop, GIMP, etc.)
- Icon generators

Make sure to create icons at multiple resolutions for best quality across different displays.
